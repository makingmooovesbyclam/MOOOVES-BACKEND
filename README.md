# Mooves

Mooves is a modern logistics and delivery management platform designed to make moving goods simple, fast, and reliable. Whether you’re sending a small package or managing bulk deliveries, Mooves connects senders, couriers, and recipients in a seamless, secure, and efficient way.

---

## 🚀 Features
- User-friendly booking – Quickly schedule pickups and deliveries.
- Real-time tracking – Track the exact location of your parcel.
- Secure payments – Safe and encrypted transaction handling.
- Multi-role access – Separate portals for users, drivers, and admins.
- Notifications – Email/SMS updates for every delivery stage.

---

## 🛠 Tech Stack
- Backend: Node.js, Express.js, MongoDB
- Frontend: React.js (or specify your frontend tech)
- Authentication: JWT-based secure login
- API Documentation: Swagger

```bash
